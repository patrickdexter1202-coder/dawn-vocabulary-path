# 晨光词径产品规格 v0.2

> 历史版本：当前完整规格请以 `english-learning-system-spec-v0.8.md` 为准。

更新日期：2026-06-30  
适用阶段：上海五四学制，2026 年秋季升六年级，短期本地试运行

## 1. 本版目标

把原有“听音跟读”扩展为完整闭环：选择词库 → 跟读 → 看中文默写英文 → 可选部分字母提示 → 记录结果 → 错词进入巩固清单。

试运行优先验证孩子是否愿意持续使用、默写难度是否合适、错词能否通过重复出现得到巩固。

## 2. 初始词库范围

### 2.1 小学基准

- 来源：工作簿“小学基线”sheet，共 498 条记录。
- 依据：义务教育英语课程标准（2022 年版）二级相关词汇，经开放词典补充原始音标和中文释义。
- 状态：范围可用于试运行；音标、中文义项和例句仍需教研复核后才能正式发布。

### 2.2 六年级词汇如何区分

“六年级词”不是“课标里不属于小学基准的所有词”，也不能只按词频或主观难度判断。正式标签必须同时具备明确的教材版本、册次、单元、可追溯证据和审核状态。

本次只纳入可交叉核实的“2024 新版牛津上海版五四制六年级上册 Unit 1 School life”词汇。公开词表 48 条来源记录中，合并 French、online、outside、comment 等重复词性记录后，形成 44 个学习词条。2026 年秋季实际版本仍需收到学生实物教材后按封面、ISBN、页码复核。

### 2.3 可选词库

| 词库 | 内容 | 本版用途 |
|---|---|---|
| 衔接混合 | 六上 Unit 1 新词优先 + 小学基准去重补充 | 默认短期试运行 |
| 小学基准 | 498 条小学基准 | 查漏补缺 |
| 六年级新词 | 44 条六上 Unit 1 可核实词 | 暑期预习 |

每轮最多呈现 30 个词。切换词库后从该库首词重新开始。

## 3. 学习流程

### 3.1 跟读

- 展示英文、音标、中文核心义和适龄原创例句。
- 支持浏览器英文语音合成，速度为 1.0x 或 0.75x。
- 确认“我跟读好了”后进入同词默写，不直接跳过。

### 3.2 默写

- 题面只给中文义项，要求填写英文。
- 可请求一次部分字母提示：每个单词显示首尾字母，中间隐藏。
- 比较时忽略大小写、句点和多余空格；短语仍要求正确词序。
- 空答案不计入考察记录。

### 3.3 结果与巩固

- 答对后可进入下一词。
- 答错时显示正确答案并加入错词清单。
- 同一错词去重保存，累计错误次数，保留最后答错时间和下次复习日期。
- 本版先提供清单查看；后续再实现按遗忘曲线自动插入复习队列。

## 4. 每日记录

以本地日期为键，记录 `tested`（考察数）、`correct`（答对数）、`wrong`（答错数）、`hinted`（提示数）和逐次作答明细。

页面顶部实时展示“已考察 N 个、错词 N 个”。个人菜单中的“学习记录”展示当日汇总，“错词巩固”展示去重后的清单。

## 5. 数据模型

```text
Word {
  id, word, normalizedAnswer, phonetic, meaning, example,
  libraries[], unitLabel, sourceType, reviewStatus
}

ProgressV1 {
  version: 1,
  daily: { [yyyy-mm-dd]: DailyRecord },
  wrongWords: { [wordId]: WrongWordRecord }
}
```

浏览器存储键：`dawn-vocabulary-progress-v1`。

## 6. 内容与版权规则

- 国家课标决定范围，指定教材决定年级、册次和单元。
- NGSL 只作高频排序与候选补充，不覆盖教材优先级。
- 开放词典字段必须保留来源并经过复核。
- 展示例句采用原创短句，不复制商业词典例句。
- 系统不分发教材扫描件；未按实物教材复核的六年级内容显示“试运行”。

## 7. 本地部署与迁移

生产网页为静态文件，可复制到另一台 Windows 电脑。迁移包用 PowerShell/.NET 自带能力启动本地 HTTP 服务，不依赖 Node.js、Python 或互联网。

地址固定为 `http://127.0.0.1:8768/`，仅本机可访问。学习记录属于浏览器本地数据，不随网页文件自动迁移；短期试运行应固定电脑和浏览器。

## 8. 非本版范围

- 登录、云同步、多孩子账号、语音识别评分和家长周报；
- 六年级全册及其他年级的完整教材词表；
- 跨电脑导入/导出学习记录；
- 正式教研审核后台。

## 9. 验收标准

- 三套词库可切换，数量与来源标签正确；
- 完成“跟读—默写—结果—下一词”；
- 提示、答对、答错均能记录；
- 错词清单去重并累计错误次数；
- 刷新后记录仍存在；
- 桌面、平板和手机核心操作可用；
- Windows 迁移包双击后可通过本地地址访问；
- 自动化测试、生产构建和浏览器验收通过。
