# 晨光词径产品规格 v0.4

> 历史版本：当前完整规格请以 `english-learning-system-spec-v0.7.md` 为准。

更新日期：2026-06-30  
适用阶段：上海五四学制，2026 年秋季升六年级，本地短期试运行

## 1. 本版范围

v0.4 在 v0.3 每日会话基础上补齐四项试运行体验：长单词安全排版、同优先级随机抽词、两层学习记录、家长密码保护删除。

## 2. 动态选词与随机规则

基础计划仍为 30 词，优先级不变：活跃高频错词 → 到期复习词 → 普通巩固词 → 当日未安排新词 → 补位词。活跃高频错词仍必须出现，可使计划超过 30 词。

每个优先级分组先随机打乱，再按必要的学习强度排序；强度相同的词保留随机顺序。因此每轮不会按字母顺序机械出题，同时不会削弱高频错词和到期复习的优先权。同一天多次开始时，优先使用当天尚未安排的新词。

补充两项交错规则：

- 最后一次已经答对、且尚未到 `nextReviewAt` 的词进入最低优先候选；只有其他候选不足时才补回。高频错词和已经到期的复习词不受此规则影响。
- 每个候选组出列时，若仍有不同首字母的单词可选，优先选择不同首字母；只有候选结构无法交错时才允许相邻同首字母。

## 3. 长单词展示

- 10–15 个字符使用紧凑字号；16 个及以上使用超紧凑字号。
- 单词区域限制最大宽度，并允许极端长词安全换行。
- 主播放按钮由最大 308px 缩至 264px，图标同步缩小，为词条保留更清晰的视觉边界。
- 桌面、平板和手机断点继续覆盖字号与按钮尺寸。

## 4. 学习记录两层结构

第一层只显示当日答对、答错数量，以及每次学习的开始时间、用时和对错小计；不直接铺开单词。

点击某次学习后，第二层展示该次全部作答：英文、中文义、正确/错误、用户填写内容、正确答案（答错时）和作答时间。仅已结束的会话允许删除，避免破坏正在学习的状态。

## 5. 家长验证删除

“学习记录”和“错词巩固”均支持单项删除，固定家长密码为 `99bill`：

- 删除学习记录：删除整次会话，并从剩余会话重新计算单词统计与错词清单。
- 删除错词：清除该词当前统计和强制巩固状态；历史会话明细保留。删除时记录截止时间，旧错误不会因日后重算重新激活；若之后再次答错，该词会重新进入巩固清单。
- 密码仅用于家庭本地操作防误删，保存在前端代码中，不属于安全级身份认证。

## 6. 本地数据

存储键仍为 `dawn-vocabulary-progress-v2`，新增可选字段：

```text
ProgressV2 {
  version: 2,
  sessions: Session[],
  wordStats: { [wordId]: WordStat },
  wrongWords: { [wordId]: WrongWordHistory },
  dismissedWrongWords: { [wordId]: deletedAt }
}
```

旧 v2 数据会自动补齐空字段，无需人工迁移。数据仍只保存在当前电脑、当前浏览器。

## 7. 验收标准

- 长单词不侵入播放区域，极端长度可安全换行。
- 同一优先级抽词不按字母排序；高频错词和到期复习优先级不变。
- 尚未到期的前序正确词优先错开；存在其他候选时不连续出现相同首字母。
- 学习记录首层只展示答对、答错与会话摘要；展开后可查全部逐词结果。
- 删除记录或错词前必须输入 `99bill`，错误密码不得删除。
- 删除会话后派生统计正确重算；删除错词后旧错误不重新激活。
- 自动化测试、生产构建、迁移包和本地 HTTP 页面验证通过。
