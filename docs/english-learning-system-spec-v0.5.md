# 晨光词径产品规格 v0.5

> 历史版本：当前完整规格请以 `english-learning-system-spec-v0.8.md` 为准。

更新日期：2026-06-30

## 1. 回车快捷操作

学习页支持使用回车执行当前主要动作：

- 跟读阶段：回车等同于“我跟读好了”，进入默写。
- 默写输入阶段：输入框中的回车仍由表单处理，用于检查答案。
- 答题结果阶段：回车等同于“进入下一个单词”。

当焦点位于输入框、按钮、链接、选择框，或个人菜单、学习记录、家长验证弹窗打开时，不触发页面级快捷操作，避免重复提交或误操作。

## 2. 低重复选词

后续轮次只使用两类单词：

1. **活跃错词**：历史上答错且尚未连续答对 3 次。高频错词必须出现；其他错词按到期时间和薄弱程度安排。若某个错词刚刚答对且尚未到复习时间，则等待到期。
2. **未学新词**：在当前词库中没有任何作答记录的词，优先选择当天尚未安排过的词。

没有错误记录的已学正确词，即使原 `nextReviewAt` 已到，也暂不进入后续轮次。系统不再为了凑满 30 词补入正确词；候选不足时允许本轮少于 30 词。

如果当前词库既没有新词，也没有待巩固错词，点击开始后停留在首页，并提示“当前词库暂时没有新词或待巩固错词”，不创建空白会话。

## 3. 随机与首字母交错

候选词先随机打乱。在同一候选组内，只要存在不同首字母的选择，就优先避开与上一词相同的首字母；仅在候选结构无法交错时允许连续同首字母。

## 4. 验收标准

- 跟读阶段和结果阶段可用回车前进。
- 默写输入框回车只提交答案，不跳过结果页。
- 纯正确词不进入任何后续轮次。
- 活跃错词仍可按错误强度与到期时间复习。
- 候选不足时返回实际数量，不以正确词补足 30 词。
- 候选耗尽时不创建空白会话。
- 同首字母在存在其他选择时不连续出现。
