# 晨光词径产品规格 v0.9

更新日期：2026-07-03
适用场景：单一家庭、GitHub Pages 在线网页、每个浏览器一份本地数据

## 1. 产品与数据边界

晨光词径是 React/Vite 静态网页，正式地址为 `https://patrickdexter1202-coder.github.io/dawn-vocabulary-path/`。GitHub Pages 只分发程序、字体和内置词库，不接收学习记录或自定义词库。

当前版本不提供账号、云数据库、GitHub 云端记录、跨设备自动同步、服务器端学习数据或原生 iOS App。相同网址在不同设备、浏览器或无痕窗口中仍是不同的数据副本。

## 2. 词库

- 内置只读词库：衔接混合 774 条、小学基准 498 条、六年级上 Unit 1-6 277 条。
- 自定义词库：支持 CSV、JSON、XLSX，保存在 IndexedDB `dawn-vocabulary-local-v1`。
- 导入必填字段为 `word`、`meaning`；可选字段与限制以 `vocabulary-import-format-v1.md` 为准。

## 3. 学习流程

选择词库 → 跟读 → 看中文默写英文 → 查看结果 → 下一词。学习页不展示例句；移动端跟读页压缩单词、释义、朗读和下一步操作，默写及结果阶段移除冗余答案占位。

动态选词优先级为活跃高频错词、到期活跃错词、其他活跃错词、未学新词。已学且没有错误历史的正确词不用于补足后续轮次。

## 4. 发音

优先请求有道在线英音；3.5 秒未开始、加载或播放失败时回退到浏览器/系统英文语音。1.0 倍和 0.75 倍速度对两条播放路径均生效；两者都失败时显示错误提示。

## 5. 会话与近 90 天记录

学习记录保存在当前浏览器 `localStorage` 的 `dawn-vocabulary-progress-v2`，包含会话日期、开始/结束时间、词库、计划词、逐次答案、提示、正误、作答时间、累计单词统计与错词状态。

“学习记录”默认显示当天，并支持：

- 前一天、后一天和原生日历选择；
- 查询当天及之前 89 天，共 90 个自然日；
- 按所选日期显示答对、答错和会话数量；
- 展开每次会话查看逐词答案与时间；
- 没有记录的日期显示明确空状态；
- 不允许选择未来日期或 90 天窗口以前的日期。

90 天窗口限制的是界面查询范围。底层旧会话不主动删除，以免破坏累计复习统计、错词状态和既有备份；完整备份仍包含浏览器中实际保存的全部会话。

删除已结束会话或错词状态需要本地家长密码 `99bill`。该密码只用于防误操作，不是账号或安全认证。

## 6. 备份与恢复

“导出完整备份”把 v2 学习记录和 IndexedDB 自定义词库写入带版本号的 JSON。恢复会覆盖当前浏览器数据，并在执行前显示备份时间、词库数与会话数。

换设备、换浏览器、清理网站数据或使用无痕模式前必须主动导出；GitHub Pages 网址本身不会携带本地数据。

## 7. 响应式与可访问性

保留 920px、680px、390px 断点、`100dvh`、安全区和至少 44px 触控目标。记录日期导航在手机端使用 44px / 弹性日期 / 44px 三列布局；按钮、日期输入和会话展开均具备可访问名称和禁用状态。

## 8. 发布与验证

- 公开仓库：`https://github.com/patrickdexter1202-coder/dawn-vocabulary-path`
- 主分支：`main`
- 自动发布：`.github/workflows/deploy-pages.yml`
- 发布前运行完整测试、Pages 子路径构建断言与 Vite 生产构建。

真实 iPhone Safari、家庭网络、有道音频失败回退、大文件导入和跨设备备份恢复仍需家庭设备长期验收。
