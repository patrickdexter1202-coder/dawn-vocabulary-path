# 晨光词径实施计划 v0.9

更新日期：2026-07-03
范围：当前浏览器近 90 天学习记录查询与项目文档同步

## 1. 产品边界

- 学习记录继续保存在当前浏览器 `localStorage`，不上传 GitHub，不增加账号或云同步。
- 学习记录入口默认显示当天，可查询当天及之前 89 天，共 90 个自然日。
- 90 天以前的原始数据不主动删除，避免破坏既有单词累计统计、错词复习与完整备份；界面只开放近 90 天查询。
- 日期切换后显示所选日期的答对、答错、会话列表与逐词明细；无记录时显示明确空状态。

## 2. 界面方向

沿用现有柔和玻璃质感，在学习记录标题下增加一条紧凑日期导航：前一天、原生日历输入、后一天。当天突出“今日”，历史日期使用完整年月日；移动端保持 44px 触控目标，日期栏可在窄屏自然换行，不引入新的视觉体系。
